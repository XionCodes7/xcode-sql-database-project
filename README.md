# Verity iOS SQL App

Cloud-integrated iOS application built with Swift and Xcode, connected to a SQL database backend using AWS RDS and RESTful APIs.

## Features

* User authentication and management
* Order and inventory tracking
* Real-time data communication via REST APIs
* Secure cloud storage and database integration

## Tech Stack

* Swift (iOS - Xcode)
* SQL (Relational Database)
* AWS RDS, S3
* REST APIs
* Git/GitHub

## Architecture

The app communicates with a backend API that handles database operations, enabling scalable and secure data management.

## Setup

1. Clone the repo
2. Open in Xcode
3. Configure API endpoint (if needed)
4. Run on simulator or device

## Notes

Sensitive credentials are not included. Configure your own database/API connection.
